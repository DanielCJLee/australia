# 台灣工程師在澳洲

台灣工程師前往澳洲找工作的百科全書。

主要以想長期居留在澳洲工作為主，其他職業也適用。

教你如何輕鬆搞定繁瑣流程，合法出國工作。

全文請至 [Wiki](https://github.com/chitsaou/australia/wiki) 閱讀。

其他國家請參考 [taiwanese-work-in](https://github.com/taiwanese-work-in)。
